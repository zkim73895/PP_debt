# Campus Jobs Platform

Платформа для поиска временной работы и стажировок в университете.

## Функции
- Просмотр вакансий
- Подача заявок
- Отслеживание статуса заявок

## Технологии
- Backend: FastAPI + SQLite
- Frontend: HTML/CSS/JS + Bootstrap 5
- Тестирование: Pytest + Postman

## Установка
1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Запустить бэкенд: `uvicorn backend.main:app --reload`
4. Открыть `frontend/index.html` в браузере
